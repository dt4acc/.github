dt4acc
======

Toolkit for building digital twins of synchrotron light sources.

Use these repositories:

- https://github.com/dt4acc/dt4acc  (runnable twin)
- https://github.com/dt4acc/dt4acc-lib  (core library)

Start with ``dt4acc`` and use ``dt4acc-lib`` as dependency.

See full documentation in: https://github.com/dt4acc/dt4acc/blob/dev/main/README_details.rst
